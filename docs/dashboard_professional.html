🔗 Markov Chain Analysis — BTC/USD & ETH/USD
Daily close-to-close return states · transition matrix · stationary distribution · next-day forecast
Each daily return is classified into one of three states using a ±1.0% band: Up (r > +band), Flat (|r| ≤ band), Down (r < −band). The transition matrix shows P(next state | current state) over each lookback window.
